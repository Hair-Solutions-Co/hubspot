# Events

Owns marketing event records and event-sync surfaces.

## Power

- Create and update marketing event records.
- Search or read event records for attendance and attribution workflows.

## Common Calls

- `GET /crm/v3/objects/0-54`
- `GET /crm/v3/objects/0-54/{recordId}`
- `POST /crm/v3/objects/0-54`
- `POST /crm/v3/objects/0-54/search`
- `PATCH /crm/v3/objects/0-54/{recordId}`
