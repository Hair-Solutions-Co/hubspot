# Programmable Emails

This folder holds HubSpot-native marketing email assets, modules, templates, and helper scripts.

## Power

- Manage reusable email modules and templates.
- Generate or sync supporting HubSpot properties through helper scripts.
- Keep programmable email assets close to the HubSpot deployment surface.

## Local Commands

- `python3 scripts/create_email_properties.py`
- Use `hs upload`, `hs watch`, or project commands depending on the deployment surface.
