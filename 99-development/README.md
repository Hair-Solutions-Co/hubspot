# Development (HubSpot analogue)

Design Manager themes, UI extension projects, and HubSpot sample apps. This folder is **`99-*`** so it sorts **after** business domains (`10`–`70`), overlays (`80`–`90`), breeze (`95`), and integrations (`98`)—matching the idea that **Development** sits alongside other HubSpot areas but is grouped last in this repo.

- **`design-manager/`** — **not tracked in this repo**; maintained as an independent Git repository. Clone into this path locally for the customer portal (theme, CMS models, Next.js app, ops scripts).
- **`hubspot-ui-cards/`** — UI extensions project.
- **`example-apps/`** — official reference repos.
