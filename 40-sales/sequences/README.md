# Sequences

Owns sequence metadata and enrollment operations.

## Power

- Read sequences.
- Enroll contacts into sequences where supported.
- Inspect enrollment state.

## Common Calls

- `GET /automation/v4/sequences`
- `GET /automation/v4/sequences/{sequenceId}`
- `POST /automation/v4/sequences/enrollments`
- `GET /automation/v4/sequences/enrollments/{enrollmentId}`
