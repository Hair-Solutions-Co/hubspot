# Meetings

Owns meeting-link metadata and meeting scheduling surfaces.

## Power

- Read meeting links and availability metadata.
- Pair booking data with CRM meeting records when required.

## Common Calls

- `GET /scheduler/v3/meetings/meeting-links`
- `GET /scheduler/v3/meetings/meeting-links/{meetingLinkId}`
- `GET /scheduler/v3/meetings/meeting-links/book/{meetingLink}`
