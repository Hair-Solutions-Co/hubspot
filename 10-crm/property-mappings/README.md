# Property Mappings

This folder owns the bridge between external field models and HubSpot property models.

## Power

- Store external-to-HubSpot field mappings.
- Separate UI mapping workflows from backend sync logic.
- Cache or materialize property definitions for mapping assistants.

## Subfolders

- `client/` - mapping UI.
- `server/` - mapping API/backend.
- `mappings/` - mapping rules and persisted relationships.
- `definitions/` - source and target field definitions.
