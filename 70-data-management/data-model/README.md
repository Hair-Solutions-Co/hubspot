# Data Model

Owns object-model structure across CRM and CMS-adjacent data.

## Power

- Manage schemas, custom objects, properties, associations, and activation-dependent object types.

## Common Calls

- `GET /crm/v3/schemas`
- `POST /crm/v3/schemas`
- `PATCH /crm/v3/schemas/{objectTypeId}`
- `GET /crm/v3/properties/{objectType}`
- `POST /crm/v3/properties/{objectType}`
