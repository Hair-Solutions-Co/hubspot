# MCP

This folder owns MCP or agent wrappers around HubSpot APIs.

## Power

- Expose HubSpot operations to agents through tool wrappers.
- Compose multiple HubSpot endpoint families behind agent-safe actions.

## Rule

- `20-mcp/` is not the source of truth for auth or schema.
- Scope ownership stays in the domain folders; MCP wraps them.
