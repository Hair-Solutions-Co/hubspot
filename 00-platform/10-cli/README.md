This folder documents the local `hs` command surface.

## Core Commands

- `hs auth`
- `hs init`
- `hs logs`
- `hs watch`
- `hs upload`
- `hs fetch`
- `hs filemanager`
- `hs hubdb`
- `hs function`
- `hs theme`
- `hs project create`
- `hs project add`
- `hs project watch`
- `hs project dev`
- `hs project upload`
- `hs project deploy`
- `hs project logs`
- `hs project list-builds`
- `hs project download`
- `hs project install-deps`
- `hs account auth`
- `hs account list`
- `hs account use`
- `hs account info`
- `hs doctor`

## Rule

- Use the CLI for HubSpot-native asset and project workflows.
- Use direct API calls for service-level automation and backend integrations.
